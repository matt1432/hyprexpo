Hyprland workspace overview

HyprExpo

Expose-style workspace selection with labels, keyboard movement, multi-monitor placement, and Lua gestures for current Hyprland.

Install Read docs

hyprpm

hyprpm add https://github.com/sandwichfarm/hyprexpo-plus

What it adds

Overview control without compositor drama.

01

Keyboard-first selection

Move focus, confirm a workspace, or quick-select by visible index or token.

02

Readable workspace labels

Show current, hover, focus, and selection states with configurable labels and borders.

03

Monitor-aware layout

Place workspace ranges per monitor with global fallbacks for mixed setups.

04

Lua gesture setup

Register touchpad gestures through `hl.plugin.hyprexpo` instead of legacy keywords.
